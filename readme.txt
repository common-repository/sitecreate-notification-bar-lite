== Description ==

This is the Really Useful Notification Bar Lite for WordPress.

What makes this so useful? Well, its flexible for a start! You can use our bar to display anything from your latest headlines, links to a special deal or even a cookie acceptance bar - we give you the tools to do whatever you want. You can even use shortcodes if you wish!

What Does The Pro Version Do?

With the Pro version you can have multiple notificaiton which scroll at the speed you set, as well as image backgrounds and more. for details see https://www.sitecreate.io/wordpress-plugins/sitecreate-really-useful-notification-bar-wordpress/

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/sc-notification-bar-lite` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. To begin using the plugin, head to Appearance > Widgets in your WP Admin and you will see the widget available.
